## v1.3.0 - 2026-05-03

- Added a single installer: `install-logi-options-switch.sh`.
- Installs enable, disable, restart, and status helper scripts.
- Creates the Only Switch `Logi Options+` switch.
- Creates the Only Switch `Restart Logi Options+` button.
- Uses `~/Documents/Scripts` as the default install folder.
- Opens the install folder in Finder after installation.
